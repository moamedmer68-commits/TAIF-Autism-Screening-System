# DATA CARD

## Datasets Used

1. **Early Autism Screening Dataset for Toddlers**  
   Used for the 12–32 months questionnaire model.

2. **Autistic Spectrum Disorder Dataset**  
   Used for the 3–12 years questionnaire screening workflow.

3. **Autism Image Dataset**  
   Used for the ResNet50 visual assessment model.

## Privacy Note

No private child images, private audio, API keys, or personal medical records should be committed to GitHub.
Only public datasets, synthetic examples, screenshots, and documentation should be shared.
